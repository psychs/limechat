LimeChat

[Japanese]

What's LimeChat ?

LimeChat is an IRC client for Mac OS X written on RubyCocoa.

Features

  • One window for multiple servers.
  • Rich keyboard shortcuts for your comfortable operations.
  • Fast and stable.

Requirements

  • Mac OS X 10.4 or 10.5
  • RubyCocoa 1.0.0

Install

OS X 10.6 Snow Leopard

  1. Download LimeChat 1.1 executable dmg.
  2. Open the downloaded dmg, and copy LimeChat.app to your Application folder.

OS X 10.5 Leopard

  1. Download LimeChat 1.1 standalone executable dmg.
  2. Open the downloaded dmg, and copy LimeChat.app to your Application folder.

OS X 10.4 Tiger

  1. Download RubyCocoa-1.0.0-OSX10.4universal.dmg, and install it.
  2. Download LimeChat 0.20 executable dmg.
  3. Open the downloaded dmg, and copy LimeChat.app to your Application folder.

Shortcut keys

Tab Complete nickname.
Ctrl+Tab
Opt+Space		 Move to the next unread channel.
Ctrl+Shift+Tab
Opt+Shift+Space		 Move to the previous unread channel.
Opt+Tab Move back to the previously selected channel.
Cmd+Up/Down
Cmd+Opt+Up/Down		 Move to the previous/next active channel.
Ctrl+Up/Down Move to the previous/next channel.
Cmd+Opt+Left/Right Move to the previous/next active server.
Ctrl+Left/Right Move to the previous/next server.
Cmd+0..9 Move to the 1st .. 10th channel in the current server.
Cmd+Ctrl+1..0 Move to the 1st .. 10th server.
Ctrl+Enter Send a notice message.
